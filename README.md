# boutPC

A simple lightweight npm package that displays basic infromation bout your computer.

## Installation

Install boutpc with npm

```bash
  npm i boutpc
```

## How to Use

- Install boutpc with npm

```bash
  npm i boutpc
```

- Create a file 'index.js', and write:

```bash
  const boutpc = require("boutpc")
  console.log(boutpc())
```

- In Terminal, write:

```bash
  node index
```
